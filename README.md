# 推しメモシェア - OshiMemo Share (Ver 2.5)

YouTube timestamped notes and template insertion tool for fan activities (推し活), video corrections, and more!
YouTubeの動画にタイムスタンプ付きのメモを残したり、定型文を挿入したりできるChrome拡張機能です。推し活や動画の修正指示などに便利です。

## 機能 / Features

- **タイムスタンプ付きメモ**: 動画の再生位置に合わせてメモを作成できます。
- **定型文挿入**: よく使うフレーズをテンプレートとして保存・挿入できます。
- **データ保存**: 作成したメモはローカルに保存されます。

## インストール方法 / Installation

1. このリポジトリをクローンまたはダウンロードします。
2. Chromeブラウザで `chrome://extensions/` を開きます。
3. 右上の「デベロッパーモード」をオンにします。
4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、ダウンロードしたフォルダ（`OshiMemoShare_Ver2.5`）を選択します。
